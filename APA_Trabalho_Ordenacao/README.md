	O arquivo 'ordenacao_3.py' referente à atividade proposta realiza o sort dos arquivos de entrada fornecidos e 
que se encontram no diretório 'instancias' utilizando como referência o arquivo 'input_names.txt'. O mesmo possui os 
nomes organizados de cada arquivo de input. 
	Conforme o programa é executado os resultados gerados por cada algoritmo de sort implementado, ou seja, os 
números organizados de forma crescente, são armazenados no diretório 'resultados' e no terminal são mostradas as etapas 
de processamento juntamente de seus tempos. Os algoritmos de sort implementados são: QuickSort, MergeSort, HeapSort, 
InsertionSort e SelectionSort. Os mesmos estão presentes no arquivo 'all_sorts.py'.
	Além disso, o sort padrão usado em python se encontra em 'default_sort.py' e também realiza o processamento das 
mesmas entradas e disponibiliza também no terminal as informações.
